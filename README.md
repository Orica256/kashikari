# かしかり帳

iPhoneのホーム画面から使える貸し借り管理アプリ（PWA）です。
App Storeへの登録不要・完全無料・オフライン対応。

## 使い方

### インストール
1. iPhoneのSafariで以下のURLを開く  
   **https://orica256.github.io/kashikari/**
2. 画面下の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ → ホーム画面にアイコンが作成される

### 機能
- **ホーム** - 貸した合計・借りた合計・差引残高のサマリーと最近の記録を表示
- **人物管理** - 貸し借りする相手を登録・編集・削除
- **取引履歴** - 全取引をフィルター（貸した／借りた／未返済／返済済み）と検索で絞り込み
- **取引詳細** - タイトル・説明・金額・日付・返済期限の管理
- **返済記録** - 一部返済・全額返済の記録
- **オフライン対応** - インターネットなしでも使用可能（データはiPhone内に保存）

## 技術情報

| 項目 | 内容 |
|------|------|
| 種別 | PWA（Progressive Web App） |
| ホスティング | GitHub Pages |
| データ保存 | localStorage（端末内） |
| 対応ブラウザ | Safari（iOS推奨） |
| 外部依存 | なし |

## ファイル構成

```
kashikari/
├── index.html    # メインHTML
├── app.js        # アプリロジック
├── style.css     # スタイル
├── sw.js         # Service Worker（オフライン対応）
├── manifest.json # PWA設定
└── icon.png      # アプリアイコン
```

## 注意事項
- データはSafariのlocalStorageに保存されます
- iPhoneを初期化するとデータが消えます
- 複数端末間のデータ同期には対応していません
